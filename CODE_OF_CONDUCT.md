# Code of Conduct

## Our Pledge

We want this project to be a respectful, constructive space for users and contributors.

## Expected behavior

- Be respectful and specific.
- Assume good intent, but prefer concrete technical feedback.
- Keep discussions focused on improving the package and repository.

## Unacceptable behavior

- Harassment, discrimination, or intimidation.
- Personal attacks or abusive language.
- Deliberate disruption of contributor workflows.

## Enforcement

Project maintainers may edit, remove, or moderate contributions and discussions that violate these expectations.
