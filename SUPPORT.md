# Support

## Getting help

- Use GitHub Issues for reproducible bugs and feature requests.
- Read the README examples and maintainer docs before filing packaging or release questions.

## Before opening an issue

Please include:

- Python version
- installation method (`uv`, wheel install, editable install)
- exact command or API call used
- minimal reproduction input
- expected vs actual result

## Project status

This is a small maintained Python package. Support is best-effort, but well-scoped bug reports and improvement proposals are welcome.
